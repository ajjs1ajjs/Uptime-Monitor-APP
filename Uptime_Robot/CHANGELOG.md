# Журнал змін

Усі помітні зміни в цьому проекті будуть задокументовані в цьому файлі.

Формат базується на [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
і цей проект дотримується [семантичного керування версіями](https://semver.org/spec/v2.0.0.html).

## [Невиданий]

### Додано
- Автоматизований конвеєр CI/CD з GitHub Actions
— Підтримка встановлення Linux через CURL
— Підтримка встановлення Linux через репозиторій APT
— Підтримка контейнера Docker через GHCR
— Інтеграція Codecov для тестового покриття
— Покращення безпеки за допомогою хешування пароля bcrypt
— Увімкнено перевірку сертифіката SSL
— Захист від SQL ін'єкції
- Конфігурація безпечних файлів cookie

### Змінено
— Перенесено з SHA-256 на bcrypt для хешування паролів
— Увімкнено перевірку SSL для запитів HTTP
— Покращено обробку помилок і журналювання
— Додано перевірку введення за допомогою Pydantic

## [1.0.0] - 2024-02-17

### Додано
- Початковий випуск
- Моніторинг роботи веб-сайту
- Відстеження терміну дії сертифіката SSL
- Багатоканальні сповіщення (Telegram, Email, Slack, Discord, Teams, SMS)
- Веб-панель інструментів
- REST API
— Аутентифікація користувача за допомогою bcrypt
- Управління сесіями
- Відстеження історії статусу
- Підтримка крос-платформи (Windows, Linux)

### Безпека
- хешування пароля bcrypt
— Захист від SQL ін'єкції
- Конфігурація безпечних файлів cookie
- Перевірка SSL включена
