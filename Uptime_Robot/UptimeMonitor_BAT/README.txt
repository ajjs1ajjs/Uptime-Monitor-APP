# Uptime Monitor - BAT Version

Ця папка містить версію для запуску через Python з BAT файлами.

## Вимоги

- Python 3.8 або вище
- Встановлені залежності (автоматично встановляться при першому запуску)

## Швидкий старт

1. Відкрийте папку `UptimeMonitor_BAT`
2. Запустіть `start.bat` - програма запуститься на порту 8000
3. Відкрийте браузер: http://localhost:8000

## Запуск на іншому порту

Запустіть `start_port.bat` з вказанням порту:
```
start_port.bat 8080
```

## Файли

- `main.py` - головний файл програми
- `ssl_checker.py` - модуль перевірки SSL сертифікатів
- `auth_module.py` - модуль автентифікації
- `requirements.txt` - залежності Python
- `start.bat` - запуск на стандартному порту 8000
- `start_port.bat` - запуск на вказаному порту

## Структура

При першому запуску автоматично створиться файл `sites.db` - база даних з інформацією про сайти.
