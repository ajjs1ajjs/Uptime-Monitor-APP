# Uptime Monitor - портативна версія

## ⚡ Швидкий старт

### На новому ПК:

1. **Встановити Python** (якщо ще не встановлено)
   - Завантажити: https://www.python.org/downloads/
   - ☑️ Важливо: поставити галочку "Add Python to PATH"

2. **Запустити** `install.bat` (від Адміністратора)
   - Встановить Python-залежності
   - Запитає порт (за замовчуванням 8080)
   - Встановить і запустить Windows Service

### Файли:

- `install.bat` - встановити як Windows Service (автозапуск)
- `build_exe.bat` - збірка EXE/інсталятора
- `create_task.ps1` - альтернативний запуск через Windows Task
- `create_task_simple.ps1` - спрощений варіант створення Task

### Особливості:

✅ **SSL перевірка** - автоматично перевіряє HTTPS сертифікати
- Сповіщення за 14 днів до закінчення
- Кожен день нагадує поки не оновлять
- 🔴 Критично: 0-3 дні
- 🟠 Важливо: 4-7 днів
- 🟡 Увага: 8-14 днів

✅ **Канали сповіщень:**
- 📱 Telegram
- 🏢 MS Teams
- 🎮 Discord
- 💬 Slack
- 📧 Email
- 📱 SMS

✅ **Windows Service:**
- Автозапуск при старті Windows
- Автоперезапуск при падінні

### Налаштування:

Веб-інтерфейс: http://localhost:8080 (або твій порт)

У веб-інтерфейсі налаштуй:
1. Способи сповіщень (Telegram, Email тощо)
2. Сайти для моніторингу

### Вимоги:

- Windows 10/11
- Python 3.8+
- Інтернет-з'єднання

### Проблеми?

Якщо не працює:
1. Перевір що Python встановлено: `python --version`
2. Запусти CMD від імені адміністратора для служби
3. Перевір порт (може бути зайнятий)
